What is this?


This is a compendium I've been working on after being inspired by FontAwesome and their incredible icon library. This library takes every obtainable item from each Zelda Game and creates a library for your access and use like a website font!

Documentation


Using "Zelda Icon's" is simple to use and implement. Using an <i> tag it implements as it's own element. Duplicate items can be differentiated by using different game tags like zimm or zioot which are "Majora's Mask" and "Ocarina of Time" respectively.

An example of different icons based on game:

<i class="zioot zi-kokiri-sword"></i>
<i class="zimm zi-kokiri-sword"></i>


The icons can also come in a variety of sizes. Ranging from a size that matches the size of the font being used but can also be set to multiples of it. Note that some icons do not handle antaliasing well and are still works in progress.

An example of different icon sizes with demonstration:
<i class="ziloz zi-magical-shield zi-lg"></i>
<i class="ziloz zi-magical-shield zi-2x"></i>
<i class="ziloz zi-magical-shield zi-3x"></i>
<i class="ziloz zi-magical-shield zi-4x"></i>
<i class="ziloz zi-magical-shield zi-5x"></i>


Along with custom sizing there is also animations and effects to be put onto these icons.
